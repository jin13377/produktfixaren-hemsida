Prints till företag

För lokaler som känns tomma eller opersonliga.

Ett enkelt print kan göra stor skillnad i en butik, ett kontor eller ett väntrum. Motivet kan bygga på företagets bransch, plats, färger eller något personligt intresse.

Målet är ett lugnt, passande motiv som gör lokalen mer välkomnande utan att kännas som massproducerad väggdekoration.
